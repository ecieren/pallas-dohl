Welcome to dohl's documentation!
==========================================

**dohl** is a python package proposed to evaluate damage in overhead lines. 

**dohl** is developed by `Eurobios <http://www.eurobios.com/>`_ and supported by `Rte-R&D <https://www.rte-france.com/>`_ *via* the OLLA project (see `ResearchGate <https://www.researchgate.net/project/OLLA-overhead-lines-lifespan-assessment>`_).
